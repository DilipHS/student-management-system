# Student Management System

A simple Java console-based application to manage student records. This project demonstrates Object-Oriented Programming (OOP) concepts, ArrayList usage, and basic CRUD (Create, Read, Update, Delete) operations.

## Features

* Add a new student
* View all students
* Search for a student by ID
* Delete a student by ID
* Menu-driven console interface

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* Eclipse IDE (or any Java IDE)

## Project Structure

```
StudentManagementSystem/
│
├── Main.java

└── README.md
```

## How to Run

### Compile

```bash
javac *.java
```

### Execute

```bash
java Main
```
